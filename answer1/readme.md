
***Question:*** Write a function that takes a list of money transfer transactions performed among a group of banks and returns the result that indicates which bank owes how much to other banks. 
### Technologies 
- Node.Js 12.18.1
- Typescript 3.9.5
### Installation
    npm install
### Run Sample Inputs
    npm start
